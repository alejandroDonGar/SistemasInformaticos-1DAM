## Usos reales de Import y Export

Estos comandos permiten automatizar el instalar programas en varios equipos.

* **Export**

    Genera un archivo JSON que guarda una lista con programas instalados. Podemos editarlo para añadir o quitar programas según nuestras necesidades.

* **Import**

    Nos permite instalar los programas que esten dentro de este archivo sin tener que hacerlo de manera manual, sino automaticamente.
