## ¿Qué es el **Moniker**?

El **Moniker** es el nombre abreviado o el apodo que tiene un programa al usar `Winget`.

| Nombre Largo       | Apodo   |
|:-------------------|:--------|
| Visual Studio Code | **`vscode`**  |
| Google Chrome      |**`chrome`**  |
| Node.js            | **`node`**    |
| Mozilla Firefox    | **`firefox`** |
| OBS Studio         | **`obs`**     |
