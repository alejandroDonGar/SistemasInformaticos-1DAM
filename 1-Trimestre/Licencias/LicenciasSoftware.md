# 🧩 Ejercicio A — Clasificación de licencias

Se deben clasificar diferentes tipos de licencias dentro de las siguientes categorías:

- **Software libre**
- **Software propietario**
- Licencias **permisivas**
- Licencias **copyleft**
- Software **gratuito**
- Software **de pago**

### Licencias analizadas

- Apache
- Apple Source
- BSD
- EPL
- Freemium
- Freeware
- GPL
- LGPL
- MIT
- MPL
- OEM
- PHP
- Shareware
- OpenSSL
- Por volumen

💡 Algunas licencias pueden aparecer en **más de una categoría**.

Por ejemplo:

- **Shareware** puede tener versión gratuita y versiones de pago.

# 🧩 Ejercicio D — Licencias Creative Commons

Se analizan diferentes licencias **Creative Commons** que aparecen en bancos de imágenes.

### Tipos de licencias

| Código | Licencia |
|----|----|
| A | Reconocimiento (BY) |
| B | Reconocimiento - Compartir Igual (BY-SA) |
| C | Reconocimiento - No Comercial (BY-NC) |
| D | Reconocimiento - No Comercial - Compartir Igual (BY-NC-SA) |
| E | Reconocimiento - Sin obra derivada (BY-ND) |
| F | Reconocimiento - No Comercial - Sin obra derivada (BY-NC-ND) |

### Casos de uso

| Situación | Licencias posibles |
|----|----|
| Modificar la imagen | A, B, C, D |
| Proyecto comercial | A, B, E |
| Proyecto no comercial con modificación | A, C |
| No citar autor | Ninguna |
| Proyecto no comercial manteniendo licencia | D |

# 📌 Conclusión

Las licencias de software y contenido digital determinan:

- cómo se puede **usar el software**
- si se puede **modificar**
- si se puede **redistribuir**
- si se puede utilizar con **fines comerciales**

Comprender estas licencias es fundamental para **desarrolladores, empresas y usuarios**, ya que define los derechos y limitaciones del uso del software.
